WORD

O Microsoft Word é um processador de texto produzido pela Microsoft Office/Microsoft 365. Foi criado por Richard Brodie para computadores IBM PC com o sistema operacional DOS em 1983.

Atalhos

1- Abrir um documento - Ctrl+O

2- Criar um novo documento - Ctrl+N

3- Criar um novo documento - Ctrl+N

4- Salvar o documento - Ctrl+S

5- Fechar o documento- Ctrl+W

6- Recortar o conteúdo selecionado para a área de transferência- Ctrl+X

7- Copiar o conteúdo selecionado para a área de transferência - Ctrl+C

8- Colar o conteúdo da Área de transferência- Ctrl+V

9- Selecionar todo o conteúdo do documento - Ctrl+A

10- Aplicar formatação em negrito ao texto- Ctrl+B

EXCEL

O Excel é um programa informático desenvolvido e distribuído pela Microsoft Corp. Trata-se de um software que permite realizar tarefas contabilísticas e financeiras graças às suas aplicações para criar e trabalhar com folhas de cálculo.

Atalhos

1- Feche uma pasta de trabalho- Ctrl+W

2- Abra uma pasta de trabalho- Ctrl+A

3- Vá para a guia Página Inicial- Alt+H

4-Salve uma pasta de trabalho- Ctrl+B

5- Copiar a seleção- Ctrl+C

6- Cole a seleção- Ctrl+V

7- Desfazer ação recente- Ctrl+Z

8- Remova o conteúdo da célula- Apagar

9-Escolha uma cor de preenchimento- Alt+C, R

10-Cortar a seleção- Ctrl+X

POWER POINT

Com PowerPoint computador, Mac ou dispositivo móvel, você pode: Criar apresentações do zero ou usar um modelo. Adicionar texto, imagens, arte e vídeos. Escolher um design profissional com o Designer do PowerPoint.

1- Criar nova apresentação- Ctrl+N

2- Adicionar um novo slide- Ctrl+M

3-Aplicar formatação em negrito ao texto selecionado- Ctrl+B

4-Abrir a caixa de diálogo Fonte- Ctrl+T

5- Recortar o texto, objeto ou slide selecionado- Ctrl+X

6- Copiar o texto, objeto ou slide selecionado- Ctrl+C

7-Colar o texto, objeto ou slide recortado ou copiado- Ctrl+V

8- Inserir um hiperlink- Ctrl+K

9-Inserir um novo comentário- Ctrl+Alt+M

10-Desfazer a última ação- Ctrl+Z
